# Conversor-de-Moedas
![Badge versão do programa](https://img.shields.io/badge/version-1.0-9cf?style=for-the-badge "Version") 

## Descrição
O projeto Conversor de Moedas consiste em um sistema de conversão de cambio desenvolvido em JAVA, criado para estudos pessoais.

## Funcionalidades
O Conversor de Moedas oferece as seguintes funcionalidades:
- Conversão de todas as moedas existentes.
- Taxa de cambio exibida.
- Sistema de log para registrar infomrações da conversão.
- Gerador de Arquivo para salvar informações.

## Exemplo de Interação do Usuario
Aqui está um exemplo de interação do usuário ao executar o programa:

```plaintext
Digite a moeda de origem (3 caracteres): eur
Digite a moeda de destino (3 caracteres): usd
Digite a quantidade: 150
Moeda de origem : EUR
Moeda de destino : USD
Quantidade : 150,00
Taxa de câmbio : 1,1866
Valor convertido : 177,99
Deseja fazer outra conversão? (1 - Sim, 2 - Não): 2
Encerrando programa...
Conversões salvas em arquivo JSON com sucesso!
```

O usuário converte 150 Euros para doláres (USD) e encerra o programa após a conversão, e com seu histórico de conversões sendo salvo em um arquivo JSON.
